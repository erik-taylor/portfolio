As a contractual developer I was tasked with creating a lightweight blog for a budding travel blogger. I contracted with the author to help outline all desired features and created a digital roadmap for the project. The answer was a mobile-first, minimal design that allows for flexible content creation and lightning-fast performance. The site is build with Enduro JS, an open source, NodeJS based static site CMS. The open-source libraries Lax.js and B-Lazy were integrated for animations/transitions and lazy-load images respectfully. A full Google Analytics integration was also included.
