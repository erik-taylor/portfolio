As a contractual developer I was tasked with replacing an out-dated Wordpress installation for a private therapy practice. I contracted with the client to help analyze the current and desired features and planned a digital roadmap for the project. The client requested the ability to share recorded audio meditations with clients in a simple an easy way, along with all the necessary content to locate and learn about their private therapy practices. The site is build with Enduro JS, an open source, NodeJS based static site CMS.
